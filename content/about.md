# About

I'm Josh Burgess — a software engineer interested in type systems, functional programming, and building reliable software.

This blog is where I write about programming topics that I find interesting or useful. Expect posts on Rust, Haskell, TypeScript, PL theory, and whatever else catches my attention.

The site itself is built with [Mythic](https://github.com/joshburgess/mythic), a static site generator I wrote in Rust.
